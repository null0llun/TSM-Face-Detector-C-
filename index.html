The implementation of a TSM Face Detection System in C++

Implementation

The TSM (Tree Structural Model) face detection system is the implementation of the “Face Detection, Pose Estimation, and Landmark Localization in the Wild” algorithm by Xiangxin Zhu and Deva Ramanan. This implementation was firstly designed for being used by embedded systems but finally it can also be used by large multiprocessors systems. The Face Detection System is implemented in C++ and there is no usage of any external library (ex. openCV, Boost) in the core of the algorithm. This gives the algorithm the ability to be used in both Windows and UNIX systems with no further changes. It also allows further improvements and alteration as it is easily readable for those who would like to use it for custom application.

As this implementation was designed for usage in embedded systems the need of reducing memory consumption was encounter. Additionally the multithreading technology was also used for multicore systems. Compared to the implementation of Hang Su, this implementation is two times faster and consumes about ten times less memory as described in the documentation

Documentation

This implementation was created as a master thesis in the Technical University of Crete by Panos Kalodimas. The Full documentation is available in the repository. The implementation offers two extra patches as referred in the documentation chapter 6 (Default Patches). The rest patches presented in chapter 8 (Alternative Patches) are not included. In the documentation the memory consumption and the execution time performance can also be found for embedded system usage.

Our implementation gives the ability of customizing the functionality of the algorithm through a set of settings and parameters that can easily be modified in the settings.xml file. The TSM System Manual describes the basic settings and how to build and run the TSM system.
